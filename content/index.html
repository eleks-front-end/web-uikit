Typography

Our framwork includes simple and easily customized typography for headings, body text, lists, and more.

Headings

All HTML headings, <h1> through <h6>, are available. .e-h1 through .e-h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

Lists

Unstyled

Remove the default list-style and left margin on list items (immediate children only). This only applies to immediate children list items, meaning you will need to add the class for any nested lists as well.

  • Lorem ipsum dolor sit amet
  • Facilisis in pretium nisl aliquet
  • Nulla volutpat aliquam velit
    • Phasellus iaculis neque
    • Ac tristique libero volutpat at
  • Aenean sit amet erat nunc
  • Eget porttitor lorem

Inline

Place all list items on a single line with display: inline-block; and some basic margin between.

  • Lorem ipsum
  • Phasellus iaculis
  • Nulla volutpat

Horizontal description

Align terms and descriptions horizontally by using our grid system’s predefined classes (or semantic mixins). For longer terms, you can optionally add a .e-text-truncate class to truncate the text with an ellipsis.

Table

Due to the widespread use of tables across third-party widgets like calendars and date pickers, we’ve designed our tables to be opt-in. Just add the base class .e-table to any <table>.

Example

Add .e-table-hover to enable a hover state on table rows within a <tbody>.

Use .e-table-striped to add zebra-striping to any table row within the <tbody>.

Add .e-table-bordered for borders on all sides of the table and cells.
