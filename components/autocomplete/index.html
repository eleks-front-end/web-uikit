Autocomplete

Usage

To apply this component, add the .ui-autocomplete class to a input element.

To enable the necessary JavaScript for the autocomplete input, you also need to add the data-ui-autocomplete attribute. Add {source:'SOURCE-URL'} to the data attribute and set the path to your autocomplete list, which needs to be formatted in JSON.
